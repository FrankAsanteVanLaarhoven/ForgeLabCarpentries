# Pre-Workshop Survey (question bank)

Send before the workshop. Keep it short (5–8 min). Use it to (a) set the right level, (b)
establish a baseline for measuring learning, and (c) track equity of access. Copy the relevant
questions into your form tool (Google Forms / Typeform / open-source equivalent).

::: note
Make demographic questions **optional** and explain why you ask (equity tracking). Never gate
attendance on answering them.
:::

## Logistics

1. Full name (for your certificate).
2. Email (for workshop communications).
3. Which session(s) are you attending? *(if multiple)*
4. Will you attend **in person** or **online**?

## Setup readiness

5. Have you completed the setup instructions? — Yes / Partially / Not yet / Need help
6. Operating system — Linux / macOS / Windows / Other
7. Do you have a working terminal, Git, and Python? — All / Some / None / Not sure
8. Will you use your own machine or the cloud environment?

## Background & confidence (baseline)

Rate your current confidence (1 = none, 5 = very confident):

9. Using the command line / shell. — 1–5
10. Using Git and GitHub. — 1–5
11. Writing a small script to automate a task. — 1–5
12. Working with ROS 2 / robot software. — 1–5  *(robotics workshops)*
13. Connecting AI models to perception/planning. — 1–5  *(AI workshops)*

## Goals & expectations

14. What do you most want to be able to do after this workshop?
15. Is there a specific project or problem you're hoping to apply this to?
16. Anything we should know to support your learning (accessibility, etc.)?

## Demographics (optional — equity tracking)

17. Career stage — Student / Early career / Mid / Senior / Other / Prefer not to say
18. Field / domain — (free text) / Prefer not to say
19. How did you hear about this workshop?
20. Anything else you'd like us to know? (optional)
